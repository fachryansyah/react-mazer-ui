## Installation
Using NPM
```bash
$ npm install react-mazer-ui --save
```

Using Yarn
```bash
$ yarn add react-mazer-ui
```

Or clone the project
```bash
$ git clone https://github.com/fachryansyah/react-mazer-ui && npm install && npm run compile
```
